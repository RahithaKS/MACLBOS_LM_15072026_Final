Request: Entra ID App Registration for Development Environment

Could you please assist with creating the App Registration by following the steps below?

Step 1: Create the App Registration
Navigate to portal.azure.com (Bosch tenant) - Microsoft Entra ID - App registrations - New registration

Fill in the following:
Name: LedgerLM
Supported account types: Accounts in this organizational directory only (Bosch – Single tenant)
Redirect URI: Platform: Web | URL: https://Genie-Finance-d.bosch.tech/api/auth/sso/microsoft/callback
Click Register

Step 2: Copy Credentials
Once registered, please share the following from the Overview page:
Application (Client) ID
Directory (Tenant) ID

Step 3: Create a Client Secret
Go to Certificates & secrets → New client secret
Description: LedgerLM SSO | Expiry: 24 months (or per Bosch policy)
Please share the secret Value immediately after creation (it is only visible once)

Step 4: Add API Permissions
Go to API permissions - Add a permission - Microsoft Graph - Delegated permissions
Select and add: openid, profile, email
Click Grant admin consent for Bosch
